!1 Funktionale Anforderungen
!2 General Workflow Requirements
 * Admin defines a Workflow
 * Admin activates defined workflow
 * User starts a new Workflow Instance
 * System informs Admin / User for his Active Workflow Instances
 * User / Admin enters Data to Workflow Instance
 * System shows Workflow Instance Data to Admin / User
 * System provides Workflow Instance Data and triggers automated Activities.
 * Admin views Workflow Statistics and Reports

Forschung, ob Liferay aus einem Formular-Eintrag einen Workflow anlegen kann, der als Anker die Mailadresse aus dem Formulareintrag übernehmen kann. 
Oder ob irgendetwas anderes, passendes in Liferay möglich wäre.
